| Executive Summary
ISC Phase 2 is the largest single non-dilutive grant available to D-Central — up to $1M over 2 years. It funds the full platform build following a successful Phase 1 PoC. Phase 2 applications must demonstrate: validated Phase 1 results, a credible commercialization plan, and a clear path to government procurement adoption. This document is a framework to be completed after Phase 1 results are available. The OC Transpo pilot contract, if executed before Phase 2 application, significantly strengthens the commercialization evidence. |
|---|


# 1. Phase 2 Readiness Criteria


| Criterion | Target Evidence at Phase 2 Application | Source |
|---|---|---|
| Phase 1 PoC accepted by federal department | Formal departmental acceptance letter from Transport Canada / Public Safety Canada / RCMP | Phase 1 completion — LOAN-CM-001 |
| Working system demonstrated in relevant environment | OC Transpo pilot contract signed and first nodes operational on bus fleet | DEPLOY-TM-001 + OC Transpo pilot agreement |
| Real enforcement revenue data | At least 3 months of citation data from OC Transpo pilot — violation volume, revenue generated, dismissal rate | OC Transpo pilot revenue reports |
| Privacy compliance demonstrated | PIPEDA Privacy Impact Assessment completed and accepted. No privacy complaints from pilot period. | REG-CM-001 + pilot period data |
| Open source community engagement | GitHub repository with 50+ stars, external contributors from Carleton/uOttawa research, published technical documentation | GitHub metrics + Mitacs partnership outputs |
| Commercial momentum | 2+ additional municipalities or communities in letter-of-intent or active negotiation | MKT-TM-GOV-002 + MKT-CM-GOV-001 |
