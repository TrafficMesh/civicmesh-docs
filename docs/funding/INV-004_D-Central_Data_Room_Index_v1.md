# 2. Data Room Security Requirements


| Platform | Docsend, Dropbox Paper with NDA gating, or Notion with access controls. Do not use Google Drive for sensitive documents without access controls. |
|---|---|
| NDA requirement | Require a signed NDA before granting access to: financial projections, SR&ED ledger, commercial agreement templates, and any document marked CONFIDENTIAL. |
| Access tracking | Use a platform that logs document views, time spent per page, and download activity. This tells you which investors are serious (reading the financials thoroughly) vs. cursory. |
| Versioning | All documents in the data room are versioned. Do not overwrite — create new versions. Investors need to know they are looking at the current version. |
