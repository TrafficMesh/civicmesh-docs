# Contributing Guide

GitHub workflow:
1. Fork repository
2. Create branch: git checkout -b feature/description
3. Make commits
4. Push: git push origin feature/description
5. Open PR

Code standards:
- Python: Black, flake8, mypy
- React: ESLint, Prettier
- Hardware: Clear docs

Testing:
- Unit tests must pass
- Linting required
- Type checking required
