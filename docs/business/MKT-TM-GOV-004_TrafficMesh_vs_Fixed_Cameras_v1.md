# 1. Side-by-Side Comparison


| Attribute | Fixed ASE Camera | TrafficMesh Mobile Node |
|---|---|---|
| Capital cost | $50,000–$150,000 per installation + civil works | $800–$2,500 per node. Installed on vehicles already owned. |
| Ongoing cost | $5,000–$15,000/year per camera (maintenance, connectivity, power) | $26–$53/month per node (NOC management, cellular, cloud) |
| Coverage | Fixed single point | Entire route, continuously, from every vehicle in the fleet |
| Vandalism risk | High — known location, easy to target | Cannot be targeted — location always changing |
| GPS avoidance | 100% — every mapping app marks the location within days of installation | Not possible — location is unpredictable |
| Deterrence effect | Displacement — drivers slow at camera, speed up immediately after | Generalized — drivers don't know which vehicle has a node |
| Violation types | Speed and red light only (fixed installation) | Bus lane, bus stop, snow route, accessible parking, expired plates, construction zone — expandable |
| Road condition data | None | Digital twin: potholes, signs, pavement, construction zones — continuously updated |
| Community revenue share | None — all revenue to government | CCSC members earn per validated citation from their nodes |
| Political resilience | Low — banned by Ontario government November 2025 | High — community ownership creates financial constituency |
| Open source / auditable | No — proprietary vendor systems | Yes — full source code published, anyone can audit |
| Data ownership | Vendor or government (varies) | City of Ottawa owns all enforcement data |
| Ontario legal status | Banned for speed enforcement (Nov 2025) | Not covered by ban — mobile, human review required |


# 2. The One Sentence Summary


| TrafficMesh mobile nodes cost less, cover more, cannot be avoided, generate more violation types, maintain the city's road infrastructure data, share revenue with the community, are open source and auditable — and are not covered by the Ontario ASE ban. |
