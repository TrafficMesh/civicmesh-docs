# Design Brief


| Opening slide | Not a title slide — a visual that captures the problem. The $38.7M number, large, on a dark background. 'Ontario banned speed cameras. Ottawa lost this. We built the replacement.' |
|---|---|
| Data visualization standards | Revenue projections: conservative and optimistic lines on same chart — never just one line. Government funding table: visual showing 70–85% funded before investor capital. |
| The cooperative slide | Show the cooperative model as a flywheel diagram, not a text list. More members → more nodes → better data → more insurance revenue → bigger discounts → more members. |
| Competitor slide | One-page matrix. Rows: Flock, Ring, ShotSpotter, Waze. Columns: mobile nodes, community revenue share, open source, Canadian compliance, cooperative governance. CivicMesh is the only row with all five checked. |
| Closing slide | Not 'thank you' — 'the next step'. Specific, named action. 'Review the IRAP approval letter and schedule a technical deep-dive.' |
| Deck length | 15 slides maximum. Appendix slides acceptable for backup but should not appear in the main flow. |
