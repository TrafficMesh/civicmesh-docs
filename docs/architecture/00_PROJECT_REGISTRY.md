# CivicMesh Project Registry

Master specification for the CivicMesh distributed traffic enforcement platform.

## Quick Facts
- **MVP Phase**: OC Transpo pilot (50 buses)
- **Timeline**: Q2-Q3 2026
- **Violation Types**: Bus lane obstruction, red light running
- **Revenue Model**: Tickets + data marketplace + MSSP

## Phases
- **Phase 0 (Now)**: MVP - OC Transpo 50 buses
- **Phase 1 (Aug)**: React UI, speeding detection, insurance API
- **Phase 2 (2027)**: City fleet, contractors, department APIs
- **Phase 3 (2027)**: Civilian app, blockchain, public dashboard

See each phase document for details.
