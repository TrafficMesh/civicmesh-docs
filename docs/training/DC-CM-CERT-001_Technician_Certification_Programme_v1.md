# 1. Certification Level Definitions


| Level | Title | Scope | Prerequisites | Assessment |
|---|---|---|---|---|
| L1 | CivicMesh Node Technician — Basic | Hardware installation (Tier 3 contractor kit), basic calibration, IP67 field procedures, cable management, OBD-II connection | None — entry level | Online exam (30 questions, 70% pass) + supervised field installation on test vehicle |
| L2 | CivicMesh Node Technician — Advanced | All L1 + Tier 1/2 permanent/seasonal installation, edge compute configuration, firmware verification, ATECC608B credential ceremony, diagnostic procedures | L1 certification + 3 months field experience | Online exam (50 questions, 75% pass) + proctored practical (full Tier 1 install with time limit) |
| L3 | CivicMesh Node Technician — Expert | All L2 + tamper investigation, forensic documentation, advanced diagnostics, MSSP NOC integration, expert witness documentation support | L2 certification + 1 year field experience + MSSP supervisor endorsement | Written case study (tamper investigation scenario) + oral assessment |


# 2. Background Check Requirements


| L1 | CPIC criminal record check (basic) — no criminal convictions related to fraud, theft, or property crime within 5 years. Results verified by accredited screening provider. |
|---|---|
| L2 | Enhanced reliability check — includes CPIC + court records + employment verification for past 5 years. Results verified by accredited screening provider. |
| L3 | Enhanced security screening — L2 equivalent + credit check + professional reference verification. Renewed annually. |
| Accredited Screeners | Mintz Global Screening, Certn, or equivalent accredited providers. Screening credential issued as part of TechnicianCredential background check sub-credential. |


# 3. Annual Renewal


| L1 Renewal | 2-hour online refresher module covering platform updates + re-exam (20 questions, 70% pass). Annual background recheck. |
|---|---|
| L2 Renewal | 4-hour online refresher + practical reassessment every 2 years. Annual background recheck. |
| L3 Renewal | 8-hour continuing education + annual oral assessment. Annual enhanced background recheck. |
| Grace Period | 30-day grace period after expiry before credential revocation. NOC dashboard alerts MSSP operator 90/60/30 days before expiry. |
