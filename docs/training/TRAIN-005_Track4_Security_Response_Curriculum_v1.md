# 1. Curriculum Modules (6 hours, $399)


| Module | Content | Duration | Assessment |
|---|---|---|---|
| Module A — CivicMesh Dispatch Integration | How CivicMesh generates dispatch events. Confidence scoring and what it means for dispatch. Priority levels. Response App installation and setup. Accepting and declining dispatch in the app. | 60 min | Practical — Response App setup and first dispatch simulation |
| Module B — Evidence Chain Obligations | Every responder action creates a signed, timestamped chain entry. What 'don't modify evidence' means in practice. Handoff documentation to police. Body cam integration. | 60 min | Written exam (15 questions) |
| Module C — Incident-Type Procedures | Detailed procedures by incident type: parking, trespassing, MeshEar events (altercation, glass break, gunshot). Decision tree for each. Escalation standards — when to call police. | 90 min | Scenario assessment — 4 incident type scenarios |
| Module D — Interaction with Law Enforcement | Your role when police arrive. What you can and cannot share on-scene. How to direct formal data requests to the CivicMesh portal. Cooperative responder rights. | 45 min | Case study — simulated police arrival scenario |
| Module E — Privacy and Anti-Surveillance | What CivicMesh data you can access as a responder. What you cannot access. Why the platform does not monitor workers. Your rights as a cooperative member. | 45 min | Written exam (10 questions) |


# 2. Prerequisites


| Security licence | Valid Ontario security guard licence (Private Security and Investigative Services Act) required before Track 4 certification. Track 4 does not replace security licensing. |
|---|---|
| Background check | Enhanced reliability check (same as L2 technician) required. Renewed annually. |
| Response App access | Granted on certification. Access to dispatch events in certified responder's jurisdiction only. |
