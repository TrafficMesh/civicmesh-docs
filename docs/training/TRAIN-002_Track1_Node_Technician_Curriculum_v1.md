# 1. L1 — Basic Node Technician (4 hours, $299)


| Module | Content | Duration | Assessment |
|---|---|---|---|
| 1A — CivicMesh Overview | Platform architecture overview at a non-technical level. What TrafficMesh, CommunityShield, MeshPlate do. Why human review matters. Role of the technician in the evidence chain. | 45 min | Embedded quiz (5 questions) |
| 1B — Hardware Fundamentals | Node hardware components. Camera types and mounting. OBD-II interface. Cellular module. Tier 3 contractor kit anatomy. IP ratings and weatherproofing basics. | 60 min | Embedded quiz (10 questions) |
| 1C — Installation Procedures | Step-by-step Tier 3 kit installation. Mount selection. Cable management. OBD-II connection. Power-up and basic verification. | 60 min | Embedded quiz (10 questions) |
| 1D — Privacy Basics | What data the node collects. On-device blurring. What NOT to do (cover camera, disconnect cellular, tamper with housing). PIPEDA summary for technicians. | 30 min | Embedded quiz (5 questions) |
| 1E — Practical Assessment | Supervised installation of Tier 3 kit on test vehicle. Must complete within 10 minutes and pass all calibration checks in Technician App. | 45 min assessment + scheduling | Pass/fail — timed installation with checklist |


# 2. L2 — Advanced Node Technician (8 hours, $499)


| Module | Content | Duration | Assessment |
|---|---|---|---|
| 2A — L1 Prerequisite Review | L1 content recap + L2 scope overview. | 30 min | Not separately assessed |
| 2B — Tier 1/2 Hardware | Permanent fleet installation hardware. Vehicle-specific brackets. Hardwired power. Seasonal deployment hardware differences. -40°C operation requirements. | 90 min | Written exam module |
| 2C — Edge Compute Configuration | Jetson Orin NX basics. Firmware verification procedure. ATECC608B key confirmation. Model version check. Connecting to node via local WiFi for diagnostics. | 90 min | Written exam module |
| 2D — Calibration and Testing | Camera alignment standards. GPS lock requirements. OBD-II speed verification. Test evidence package generation and review. Credential chain verification in Technician App. | 90 min | Written exam module |
| 2E — Fault Diagnosis | Common fault types and their symptoms. Diagnostic procedure tree. NOC escalation decision matrix. Parts replacement vs. return-to-base decision. | 60 min | Case study — 3 fault diagnosis scenarios |
| 2F — Practical Assessment | Full Tier 1 installation on test fleet vehicle (bus or truck). Must complete within 60 minutes. All calibration standards must be met. InstallationCredential successfully issued. | 2 hours assessment | Pass/fail — timed with full credential issuance |


# 3. L3 — Expert Node Technician (16 hours, $799)


| Module | Content | Duration | Assessment |
|---|---|---|---|
| 3A — L2 Prerequisite + Advanced Hardware | Advanced hardware architecture. ATECC608B deep dive. Secure boot process. Firmware reproducible build verification. | 2 hours | Written exam |
| 3B — Tamper Investigation | Tamper event classification. Physical inspection methodology. Forensic documentation standards. Chain of custody for the investigation itself. Evidence preservation. | 3 hours | Case study — simulated tamper investigation |
| 3C — Advanced Diagnostics | Firmware diagnostic commands. Memory and CPU profiling. Cellular diagnostic logs. GPS signal troubleshooting. Camera ISP calibration. | 3 hours | Practical — live node diagnostic session |
| 3D — Expert Witness Preparation | How evidence packages are used in court. What defence counsel will challenge. How to prepare documentation that withstands legal scrutiny. Role of the L3 technician in legal proceedings. | 2 hours | Oral assessment — mock cross-examination scenario |
| 3E — MSSP NOC Integration | How technician field work integrates with NOC monitoring. Escalation protocols. Remote diagnostic cooperation with NOC operators. Fleet health reporting. | 2 hours | Written exam |
| 3F — Practical Assessment | Demonstrate all L3 competencies across: tamper investigation, advanced diagnostic, forensic documentation, and live NOC coordination. | 4 hours | Pass/fail — comprehensive practical |
